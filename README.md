# perceptron-rosenblatt-ru-5240251
Projeto acadêmico da disciplina Inteligência Artificial (Uninter). Implementação de um Perceptron de Rosenblatt com 7 entradas, treinado para classificar padrões em relação ao RU 5240251. Inclui planilha com 50 amostras, registro completo do treinamento supervisionado e validação dos resultados.
